## 第二部分：条款差距分析表

### 逐条对比：新版 vs 旧版（2012 版）

| 序号 | 条款 | 2012 版要求 | 2026 新版要求 | 差距程度 | 企业现状自查问题 |
|------|------|------------|-------------|---------|----------------|
| **1** | 4.1 组织所处环境 | 未明确要求 | 确定 EMS 范围时，须明确纳入**气候变化**因素的考量 | ⭐⭐⭐ 高 | 现有环评是否包含碳排放/气候风险评估？ |
| **2** | 4.1 组织所处环境 | 未明确要求 | 须纳入**生物多样性**及**全生命周期**因素 | ⭐⭐⭐ 高 | 环评是否覆盖供应链全生命周期？生物多样性是否评估？ |
| **3** | 5.1 领导作用 | 要求最高管理者支持 | 从"支持"→"**履行合规义务**"；强调自然资源保护职责 | ⭐⭐ 中 | 最高管理者是否书面承诺了合规义务和资源保护责任？ |
| **4** | 6.1 策划—风险与机遇 | 风险识别与应对 | 新增**"结构化变更管理"**条款；将策划拆分为两步：①识别风险与机遇 ②规划应对行动 | ⭐⭐⭐ 高 | 现有变更管理是否有结构化流程？是否区分了风险识别与应对规划？ |
| **5** | 6.1 策划—应急准备 | 应急准备和响应 | 应急准备须与**风险规划直接关联**；明确区分**"紧急运行"与"异常运行"**的管理要求 | ⭐⭐⭐ 高 | 应急预案是否与风险评估挂钩？紧急运行与异常运行是否分别管理？ |
| **6** | 8.1 运行控制—外包 | 管控外包过程 | 改为"**外部提供的过程**"；运营控制范围延伸至**供应商全链条** | ⭐⭐⭐ 高 | 供应商管理程序是否覆盖了所有外部提供的过程？是否定期审核？ |
| **7** | 9.1 绩效评价 | 监视、测量、分析 | 从"关注合规性"→"**明确评估环境绩效**"；须建立可量化的绩效指标 | ⭐⭐⭐ 高 | 现有指标体系是否可量化？是否包含环境绩效而非仅合规性？ |
| **8** | 9.2 内部审核 | 内审程序 | 内审须**提前确定具体审核目标**；重新规范管理评审的内容与形式 | ⭐⭐ 中 | 内审方案是否明确了具体目标？管理评审输入是否完整？ |
| **9** | 10 持续改进 | 持续改进 | 新增对改进措施有效性的**验证要求** | ⭐⭐ 中 | 改进措施完成后是否验证了有效性？是否有闭环记录？ |

## 第四部分：换版执行行动清单

### Phase 1｜紧急启动（4月，本周开始）

- [ ] **1.1** 成立换版工作组（管理者代表 + 各部门负责人 + 内审员）
- [ ] **1.2** 组织核心人员研读新版《环境管理体系认证规则》全文
- [ ] **1.3** 对照第二部分差距分析表，完成初步自查
- [ ] **1.4** 联系认证机构，确认换版审核排期及所需材料
- [ ] **1.5** 向全员发布换版通知，制定总体计划

### Phase 2｜文件修订（4月—5月）

- [ ] **2.1** 修订《环境管理手册》：补充气候变化、生物多样性、全生命周期相关内容
- [ ] **2.2** 新增 / 修订《风险与机遇管理程序》：建立结构化变更管理流程
- [ ] **2.3** 修订《应急准备与响应程序》：应急与风险评估挂钩；区分紧急运行与异常运行
- [ ] **2.4** 修订《供应商管理程序》：覆盖所有外部提供的过程，建立供应商绩效评价机制
- [ ] **2.5** 修订《环境绩效监视与测量程序》：建立可量化的环境绩效指标（非仅合规）
- [ ] **2.6** 修订《内审程序》：内审方案须提前明确具体审核目标
- [ ] **2.7** 修订《管理评审程序》：规范管理评审输入内容与形式
- [ ] **2.8** 修订《持续改进程序》：增加改进措施有效性验证闭环要求
- [ ] **2.9** 更新所有相关表单、记录模板

### Phase 3｜培训实施（5月）

- [ ] **3.1** 管理层专项培训：最高管理者合规义务、资源保护职责（条款 5.1）
- [ ] **3.2** 内审员换版培训：新版审核要点、现场审核技巧
- [ ] **3.3** 全员宣贯：新版变化要点及对日常工作的影响
- [ ] **3.4** 供应链管理人员培训：供应商管控新要求

### Phase 4｜内部试运行（5月—6月）

- [ ] **4.1** 新版文件试运行，收集运行中发现的问题
- [ ] **4.2** 组织一次内部审核（按新版要求，提前明确审核目标）
- [ ] **4.3** 召开管理评审会议（按新版要求的输入内容）
- [ ] **4.4** 对试运行中发现的问题进行整改，形成闭环记录
- [ ] **4.5** 确认所有整改措施的有效性

### Phase 5｜认证换版审核（6月—7月，结合年度监督审核）

- [ ] **5.1** 向认证机构提交换版申请，准备以下材料：
  - 新版体系文件（环境管理手册 + 程序文件 + 记录表单）
  - 差距分析记录
  - 内审报告 + 管理评审报告
  - 变更说明（每一条差距对应的修改内容）
- [ ] **5.2** 配合认证机构完成现场审核
- [ ] **5.3** 完成不符合项整改（如有）
- [ ] **5.4** 获取新版认证证书，记录有效期

### Phase 6｜常态化运行（换版后持续）

- [ ] **6.1** 按新版规则常态化运行 EMS
- [ ] **6.2** 迎接年度监督审核
- [ ] **6.3** 每周期关注认监委 / 市场监管总局最新监管动态

## 第五部分：监管红线提醒

> 以下行为将面临市场监管总局重点查处（依据市监认证发〔2026〕25 号）：

| 🚫 禁止行为 | 说明 |
|------------|------|
| 超范围认证 | 认证范围不得超出认可批准范围 |
| 伪造审核记录和档案 | 审核须全程真实记录，不得到现场即出报告 |
| 买卖证书 / "包过" | 坚决取缔以认证咨询"一条龙"、"包过"揽客的机构 |
| 审核时间缩水 | 须保证标准规定的最小审核时间 |
| 审核组能力不符 | 审核组长及组员须具备相应专业能力 |
| 减少/遗漏审核程序 | 不得跳步、漏步 |

> ⚠️ 一旦获证组织体系不符合要求且未处置，将**倒查认证机构责任**，认证机构承担连带责任。
