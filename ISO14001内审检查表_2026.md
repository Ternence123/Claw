> 适用范围：内部审核 / 换版审核前自查

## 使用说明

- **判定栏**：✅ 符合 / ❌ 不符合 / ⚠ 部分符合 / N/A 不适用
- **证据栏**：记录支撑结论的文件、记录、访谈对象
- **整改栏**：如判定为不符合或部分符合，须填写整改措施及完成期限
- 每条检查要点均可在"备注"栏补充说明

## 条款 4：组织所处环境

| 序号 | 检查要点 | 判定 | 证据 | 整改措施 | 完成日期 |
|------|---------|------|------|---------|---------|
| 4.1 | 组织是否已识别与其环境管理体系相关联的**内外部因素**？ | | | | |
| 4.2 | 确定 EMS 范围时，是否明确纳入了**气候变化**因素的考量？ | | | | |
| 4.3 | 确定 EMS 范围时，是否明确纳入了**生物多样性**因素？ | | | | |
| 4.4 | 确定 EMS 范围时，是否考虑了**产品/服务全生命周期**的环境因素？ | | | | |
| 4.5 | 组织是否识别了**相关方**及其**需求和期望**，并确定了合规义务？ | | | | |
| 4.6 | EMS 范围文件是否经**最高管理者**批准并形成记录？ | | | | |

## 条款 5：领导作用

| 序号 | 检查要点 | 判定 | 证据 | 整改措施 | 完成日期 |
|------|---------|------|------|---------|---------|
| 5.1 | **最高管理者**是否对 EMS 的建立、实施、保持和持续改进作出**书面承诺**？ | | | | |
| 5.2 | 最高管理者承诺是否明确包含**履行合规义务**和**自然资源保护**的职责？ | | | | |
| 5.3 | 最高管理者是否确保了环境**方针**的建立，并在组织内得到**沟通**？ | | | | |
| 5.4 | **管理者代表**是否已明确指定并授权？ | | | | |
| 5.5 | 管理者代表是否具备足够的**权限**来协调 EMS 相关工作？ | | | | |

## 条款 6：策划

| 序号 | 检查要点 | 判定 | 证据 | 整改措施 | 完成日期 |
|------|---------|------|------|---------|---------|
| 6.1 | 组织是否已建立**结构化的变更管理流程**（新增要求）？ | | | | |
| 6.2 | 策划是否分两步执行：①识别风险与机遇 ②规划应对行动？ | | | | |
| 6.3 | **风险评估**是否覆盖了环评因素、合规义务、气候变化影响？ | | | | |
| 6.4 | 识别的风险与机遇是否制定了**应对措施**并纳入体系规划？ | | | | |
| 6.5 | **应急准备与响应**是否与**风险规划直接关联**（新增要求）？ | | | | |
| 6.6 | 组织是否已区分并分别管理"**紧急运行**"与"**异常运行**"（新增要求）？ | | | | |
| 6.7 | 应急预案是否经**最高管理者**批准并定期演练？ | | | | |

## 条款 7：支持

| 序号 | 检查要点 | 判定 | 证据 | 整改措施 | 完成日期 |
|------|---------|------|------|---------|---------|
| 7.1 | 组织是否为 EMS 提供了必要的**资源**（人、财、物、技术）？ | | | | |
| 7.2 | **能力管理**：从事影响环境绩效工作的人员是否具备必要的能力？是否有培训记录？ | | | | |
| 7.3 | 管理层及内审员是否已完成**新版标准专项培训**（换版必查）？ | | | | |
| 7.4 | 环境**方针**是否在员工入职/上岗培训中得到传达？ | | | | |
| 7.5 | 组织内部是否建立了有效的**沟通机制**（上下/横向/外部）？ | | | | |
| 7.6 | EMS 所需的文件化信息是否均已**受控管理**（编制/审批/发布/变更）？ | | | | |

## 条款 8：运行

| 序号 | 检查要点 | 判定 | 证据 | 整改措施 | 完成日期 |
|------|---------|------|------|---------|---------|
| 8.1 | 运行策划是否已**覆盖所有合规义务**要求？ | | | | |
| 8.2 | **供应商/承包商管理**是否已从"外包"扩展至"**外部提供的过程**"全链条？ | | | | |
| 8.3 | 是否有对**供应商**的环境绩效要求及定期**评价机制**？ | | | | |
| 8.4 | 是否识别并管控了所有可能导致**环境影响**的采购物品/服务？ | | | | |
| 8.5 | 应急准备与响应程序是否已按新版要求与**风险规划**挂钩？ | | | | |
| 8.6 | 是否定期开展**应急演练**并记录结果？ | | | | |

## 条款 9：绩效评价

| 序号 | 检查要点 | 判定 | 证据 | 整改措施 | 完成日期 |
|------|---------|------|------|---------|---------|
| 9.1 | 组织是否建立了**可量化的环境绩效指标**体系（新增要求）？ | | | | |
| 9.2 | 环境绩效指标的评估是否已从"**仅关注合规性**"升级为"**评估实际绩效**"？ | | | | |
| 9.3 | **监测与测量**是否覆盖了所有重要环境因素？是否有记录？ | | | | |
| 9.4 | **合规性评价**：是否定期（至少每年一次）评价了对适用法律法规的遵循情况？ | | | | |
| 9.5 | **内审方案**是否在审核前明确了**具体审核目标**（新增要求）？ | | | | |
| 9.6 | 内审员是否具备相应的**专业能力**？内审记录是否完整？ | | | | |
| 9.7 | 管理评审是否按新版要求提供**完整的输入信息**？ | | | | |
| 9.8 | **管理评审输出**是否包含对环境方针、目标、指标的修订决定及资源需求？ | | | | |
